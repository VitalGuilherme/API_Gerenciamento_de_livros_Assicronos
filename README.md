# API de gerenciamento de livros (FastAPI Assíncrona)

Este projeto desenvolvida para o exercício focada na implementação de endpoints assíncronos utilizando **FastAPI**, **Pydantic** e o padrão **async/await**. Lembrando que, devido o uso de um banco de dados, via lista armazenando dados localmente na memória, o "await" acaba se tornando oculto. As operações CRUD são instantâneas para o processador.

# Tecnologias utilizadas:
* Python 3.13.3
* FastAPI 0.0.21 
* Uvicorn 0.40.0
* Pydantic

# Pré-requesitos
Antes de começar, você precisará ter o Python instalado em sua máquina. Recomenda-se o uso de um ambiente virtual.
